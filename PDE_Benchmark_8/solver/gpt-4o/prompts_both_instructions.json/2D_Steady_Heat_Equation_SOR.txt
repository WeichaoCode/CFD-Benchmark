Below is the Python code to solve the given steady-state heat equation using the Successive Over-Relaxation (SOR) method. The code initializes the temperature field, applies the SOR update iteratively, enforces the Dirichlet boundary conditions, checks for convergence, and finally saves the temperature distribution to a `.npy` file.

```python
import numpy as np
import matplotlib.pyplot as plt

# Parameters
Lx, Ly = 5.0, 4.0  # Domain size
dx, dy = 0.05, 0.05  # Grid spacing
nx, ny = int(Lx/dx) + 1, int(Ly/dy) + 1  # Number of grid points
omega = 1.5  # Relaxation factor
tolerance = 1e-4  # Convergence criterion

# Initialize temperature field
T = np.zeros((ny, nx))

# Apply Dirichlet boundary conditions
T[:, 0] = 10.0  # Left boundary
T[:, -1] = 40.0  # Right boundary
T[0, :] = 20.0  # Bottom boundary
T[-1, :] = 0.0  # Top boundary

# SOR method
beta = dx / dy
converged = False
iteration = 0

while not converged:
    T_old = T.copy()
    for j in range(1, ny-1):
        for i in range(1, nx-1):
            T[j, i] = omega * (T_old[j, i+1] + T[j, i-1] + beta**2 * (T_old[j+1, i] + T[j-1, i])) / (2 * (1 + beta**2)) + (1 - omega) * T_old[j, i]
    
    # Enforce boundary conditions
    T[:, 0] = 10.0
    T[:, -1] = 40.0
    T[0, :] = 20.0
    T[-1, :] = 0.0

    # Check for convergence
    max_diff = np.max(np.abs(T - T_old))
    if max_diff < tolerance:
        converged = True
    iteration += 1

# Save the final temperature field
np.save('temperature_distribution.npy', T)

# Plot the temperature distribution
plt.figure(figsize=(8, 6))
plt.contourf(np.linspace(0, Lx, nx), np.linspace(0, Ly, ny), T, 20, cmap='hot')
plt.colorbar(label='Temperature (Â°C)')
plt.title('Temperature Distribution')
plt.xlabel('x')
plt.ylabel('y')
plt.show()
```

### Explanation:
- **Initialization**: The temperature field `T` is initialized to zero, and the boundary conditions are applied.
- **SOR Iteration**: The temperature field is updated using the SOR formula. The boundary conditions are reapplied after each iteration.
- **Convergence Check**: The loop continues until the maximum change in temperature between iterations is less than the specified tolerance.
- **Output**: The final temperature distribution is saved to a `.npy` file and visualized using a contour plot.
