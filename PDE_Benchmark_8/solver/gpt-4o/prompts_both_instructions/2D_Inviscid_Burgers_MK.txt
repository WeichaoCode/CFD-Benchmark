```python
import numpy as np

# Domain parameters
nx, ny = 151, 151
x = np.linspace(0, 2, nx)
y = np.linspace(0, 2, ny)
dx = x[1] - x[0]
dy = y[1] - y[0]

# Time parameters
nt = 300
sigma = 0.2
dt = sigma * min(dx, dy) / 2

# Initialize velocity fields
u = np.ones((ny, nx))
v = np.ones((ny, nx))

# Initial conditions
u[int(0.5 / dy):int(1 / dy + 1), int(0.5 / dx):int(1 / dx + 1)] = 2
v[int(0.5 / dy):int(1 / dy + 1), int(0.5 / dx):int(1 / dx + 1)] = 2

# MacCormack method
for n in range(nt):
    # Predictor step
    u_star = np.copy(u)
    v_star = np.copy(v)
    
    u_star[1:-1, 1:-1] = (u[1:-1, 1:-1] - dt * (
        u[1:-1, 1:-1] * (u[1:-1, 1:-1] - u[1:-1, :-2]) / dx +
        v[1:-1, 1:-1] * (u[1:-1, 1:-1] - u[:-2, 1:-1]) / dy))
    
    v_star[1:-1, 1:-1] = (v[1:-1, 1:-1] - dt * (
        u[1:-1, 1:-1] * (v[1:-1, 1:-1] - v[1:-1, :-2]) / dx +
        v[1:-1, 1:-1] * (v[1:-1, 1:-1] - v[:-2, 1:-1]) / dy))
    
    # Corrector step
    u[1:-1, 1:-1] = 0.5 * (u[1:-1, 1:-1] + u_star[1:-1, 1:-1] - dt * (
        u_star[1:-1, 1:-1] * (u_star[2:, 1:-1] - u_star[1:-1, 1:-1]) / dx +
        v_star[1:-1, 1:-1] * (u_star[1:-1, 2:] - u_star[1:-1, 1:-1]) / dy))
    
    v[1:-1, 1:-1] = 0.5 * (v[1:-1, 1:-1] + v_star[1:-1, 1:-1] - dt * (
        u_star[1:-1, 1:-1] * (v_star[2:, 1:-1] - v_star[1:-1, 1:-1]) / dx +
        v_star[1:-1, 1:-1] * (v_star[1:-1, 2:] - v_star[1:-1, 1:-1]) / dy))
    
    # Apply Dirichlet boundary conditions
    u[:, 0] = 1
    u[:, -1] = 1
    u[0, :] = 1
    u[-1, :] = 1
    
    v[:, 0] = 1
    v[:, -1] = 1
    v[0, :] = 1
    v[-1, :] = 1

# Save the final solution
np.save('u.npy', u)
np.save('v.npy', v)
```

This code implements the MacCormack method to solve the given PDE problem. It initializes the velocity fields with the specified initial conditions, applies the predictor-corrector steps for each time step, and enforces the Dirichlet boundary conditions. Finally, it saves the final velocity fields `u` and `v` as `.npy` files.
